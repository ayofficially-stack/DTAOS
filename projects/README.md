# Purpose

Projects는 DTAOS의 실제 업무 공간이다.

모든 프로젝트는 동일한 표준 구조를 사용한다.

Project는 DTAOS의 최상위 실행 단위이며,

Knowledge, Memory, Engine, Workflow를 연결하는 중심 객체이다.

# Rules

1. 모든 프로젝트는 P000_Template를 기반으로 생성한다.
2. Project ID는 중복될 수 없다.
3. 하나의 프로젝트는 하나의 Goal을 가진다.
4. 공통 지식은 Knowledge Layer에 저장한다.
5. 프로젝트 전용 정보만 Project 내부에 저장한다.
