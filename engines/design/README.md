# Design Engine

Version: 2.0

---

# Purpose

Design Engine은 문서의 시각적 완성도를 객관적인 기준으로 분석한다.

주관적인 디자인 의견을 제시하는 것이 아니라,

재현 가능한 기준으로 Visual Quality를 평가한다.

---

- PPT
- PDF

- Brochure

- Report

# Execution Steps

## STEP 1

Grid Analysis

확인 항목

- Column
- Alignment
- Margin
- Balance

---

## STEP 2

Visual Hierarchy

확인 항목

- Primary Message
- Secondary Message
- Reading Order
- Focus Point

---

## STEP 3

Typography

확인 항목

- Font Consistency
- Font Size
- Font Weight
- Readability

---

## STEP 4

Spacing

확인 항목

- Whitespace
- Padding
- Grouping
- Rhythm

---

## STEP 5

Color System

확인 항목

- Brand Consistency
- Contrast
- Accent Color
- Accessibility

---

## STEP 6

Information Density

확인 항목

- Text Amount
- Image Balance
- Scan Time
- Cognitive Load

---

## STEP 7

Call To Action

확인 항목

- CTA 존재 여부
- 위치
- 가시성
- 행동 유도

# Failure Conditions

다음 조건 중 하나라도 만족하면 FAIL

- Grid 없음
- Hierarchy 불명확
- Font 일관성 없음
- 여백 부족
- CTA 없음
- 정보 과다
- 브랜드 일관성 부족
