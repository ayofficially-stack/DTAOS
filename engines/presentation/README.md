# Presentation Engine

Version: 2.0

---

# Purpose

Presentation Engine은 문서의 전달력과 설득력을 분석한다.

이 Engine은 디자인의 미적 완성도를 평가하지 않는다.

또한 사업성을 직접 평가하지도 않는다.

Presentation Engine은

"사용자가 얼마나 빠르게 이해하고 행동할 수 있는가?"

를 분석한다.

---

# Execution Steps

## STEP 1

Audience Identification

확인 항목

- Target Audience
- Decision Maker
- Technical Level
- Expected Reading Time

---

## STEP 2

Core Message

확인 항목

- 핵심 메시지 존재 여부
- 첫 10초 이해 가능 여부
- Message Consistency
- One Sentence Summary

---

## STEP 3

Story Flow

확인 항목

- Introduction
- Problem
- Solution
- Evidence
- Benefit
- Next Step

---

## STEP 4

Reading Flow

확인 항목

- Reading Order
- Slide Transition
- Cognitive Load
- Information Priority

---

## STEP 5

Evidence Mapping

확인 항목

- Claims
- Supporting Data
- Customer Cases
- Certification
- References

---

## STEP 6

Call To Action

확인 항목

- Next Step
- Contact
- PoC Proposal
- Meeting Request

# Failure Conditions

다음 조건 중 하나라도 만족하면 FAIL

- 대상 독자가 명확하지 않음
- 핵심 메시지가 불분명
- 스토리 흐름 단절
- 근거 부족
- CTA 없음
- 읽는 순서가 혼란스러움
