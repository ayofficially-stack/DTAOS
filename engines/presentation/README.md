# Presentation Engine

The Presentation Engine converts strategy, technology, and business logic into slide-level communication.

## Purpose

Build or review PPT, sales decks, one-page materials, white-label sheets, and pitch narratives.

## Input

- Objective
- Audience
- Core message
- Supporting evidence
- Visual constraints
- Desired output format

## Slide Logic

Each slide should answer one question:

```text
What should the viewer understand, believe, or do after seeing this slide?
```

## Review Criteria

| Criterion | Review question |
|---|---|
| Slide role | Does this slide have one clear job? |
| Message hierarchy | Is the main point visible first? |
| Evidence | Is proof visible without overloading the page? |
| Flow | Does the slide sequence build a persuasive argument? |
| Format | Is the output suitable for editable PPT when final? |

## Default Reviewer Selection

Use Adaptive Persona Engine reviewers:

- P05 Industrial Design Director
- P06 Presentation Reviewer
- P07 Investor / Business Developer, if strategy or growth matters
- P10 Red Team

## Output Standard

Return:

```text
1. Slide objective
2. Core message
3. Layout structure
4. Suggested copy
5. Visual direction
6. Review concerns
```

## DTAOS Presentation Memory

- HTML may be used as an editorial prototype.
- Final sales or external presentation materials should preferably be editable PPT.
- Design should be professional, premium, and restrained.
- Avoid decorative icons unless they clarify meaning.
