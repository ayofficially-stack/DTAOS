# Knowledge Routing

> Define how AI locates and loads technical knowledge within the Graphene Platform.

---

# Purpose

The purpose of Knowledge Routing is to identify the minimum set of documents required to answer a technical question.

The objective is to maximize accuracy while minimizing unnecessary context loading.

---

# Philosophy

Do not read everything.

Read only what is required.

One Question

↓

One Knowledge Path

↓

One Answer

---
