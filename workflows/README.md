# DTAOS Workflow Manager

Version: 1.0

---

## Purpose

Workflow는 Engine이 작업을 수행하는 순서를 정의한다.

Dispatcher가 Engine을 선택하면

Engine은 반드시 Workflow를 따라야 한다.

Workflow는

- 일관성
- 재현성
- 품질

을 보장한다.
