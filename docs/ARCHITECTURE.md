# Mission

DTAOS exists to maximize the value of human time.

The objective is not to use AI more.

The objective is to allow humans to spend more time on decisions that only humans can make.

---

# Vision

Build an AI Operating System for DeepTech Companies.

DTAOS must become a reusable operating system that can be applied to any technology-based company.

---

---

# Design Principles

## 1. Single Source of Truth

GitHub is the master repository.

Google Drive stores only deliverables.

---

## 2. Knowledge First

AI cannot perform good work without understanding the business.

Knowledge must exist before Skills.

---

## 3. Skills before Prompts

Prompts execute.

Skills think.

Skills define how work is performed.

---

## 4. AI Agnostic

No dependency on a single AI.

Every Skill should work with

- ChatGPT
- Claude
- Gemini

Future AI engines can be added without changing the Skill.

---

## 5. Build → Test → Improve

Never seek perfection before testing.

Every Skill starts as v0.1.

Every real project improves the Skill.
